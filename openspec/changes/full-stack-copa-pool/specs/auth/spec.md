## ADDED Requirements

### Requirement: Frictionless Login UI
The system SHALL display a login form with an input for "Nome de Guerra" (username) and an optional "PIN" (4 digits). It MUST have a prominent "Entrar no Bolão" button.

#### Scenario: User logs in successfully
- **WHEN** the user enters a valid "Nome de Guerra" and clicks "Entrar no Bolão"
- **THEN** the system logs the user in and navigates to the Daily Matches screen.
